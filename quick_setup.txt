# 빠른 시작 가이드 (5분 안에 설치하기)

## 1단계: 프로젝트 준비 (1분)

```bash
# 폴더 생성 및 이동
mkdir retirement-planning-mcp
cd retirement-planning-mcp

# 가상환경 생성
python -m venv venv

# 가상환경 활성화
source venv/bin/activate  # macOS/Linux
# 또는
venv\Scripts\activate     # Windows
```

## 2단계: 의존성 설치 (1분)

```bash
pip install mcp pydantic numpy
```

## 3단계: 서버 파일 생성 (2분)

```bash
mkdir -p mcp_server_jeoklip
```

## 4단계: Claude Desktop 설정 (1분)

### macOS

```bash
# 설정 파일 열기
open ~/Library/Application\ Support/Claude/claude_desktop_config.json
```

### Windows

```bash
# 설정 파일 열기
notepad %APPDATA%\Claude\claude_desktop_config.json
```

### 설정 내용 추가

```json
{
  "mcpServers": {
    "jeoklip": {
      "command": "python",
      "args": ["-m", "mcp_server_jeoklip"],
      "cwd": "/절대경로/retirement-planning-mcp"
    },
    "tooja": {
      "command": "python",
      "args": ["-m", "mcp_server_tooja"],
      "cwd": "/절대경로/retirement-planning-mcp"
    },
    "inchul": {
      "command": "python",
      "args": ["-m", "mcp_server_inchul"],
      "cwd": "/절대경로/retirement-planning-mcp"
    }
  }
}
```

⚠️ `/절대경로/retirement-planning-mcp`를 실제 경로로 변경!

**현재 경로 확인:**
```bash
pwd  # macOS/Linux
cd   # Windows
```

## 5단계: 테스트 및 실행

### 서버 작동 확인

```bash
# 각 서버 개별 테스트
python -m mcp_server_jeoklip  # Ctrl+C로 종료
python -m mcp_server_tooja    # Ctrl+C로 종료
python -m mcp_server_inchul   # Ctrl+C로 종료
```

오류 없이 대기 상태가 되면 성공!

### Claude Desktop 재시작

1. Claude Desktop 완전 종료
2. 다시 시작
3. 새 대화 시작

## 사용해보기

Claude Desktop에서 다음과 같이 질문:

```
안녕하세요! 은퇴 준비를 시작하려고 합니다.
40세이고 65세에 은퇴하고 싶은데,
현재 자산이 1억원입니다.

얼마나 저축해야 하나요?
```

## 문제 해결

### "ModuleNotFoundError" 오류

```bash
pip install mcp pydantic numpy
```

### 서버가 Claude에 안 보임

1. JSON 문법 확인 (쉼표, 따옴표)
2. 경로가 절대 경로인지 확인
3. Claude Desktop 완전 재시작

### 경로 문제

```bash
# 현재 디렉토리 절대 경로
pwd               # macOS/Linux
echo %CD%         # Windows
```
